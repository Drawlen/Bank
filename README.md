# Банковские счета

Учебный проект по управлению банковскими счетами клиентов.

## Функционал
- Управление клиентами
- Управление счетами
- Приходные и расходные операции
- Авторизация пользователей

## Установка
1. Импортировать bank_sys.sql в MySQL
2. Настроить подключение в dbconnect.php
3. Зайти с логином: admin, пароль: admin123
